# 🌿 Stone Feng Shui Management

## 📌 1. Project Topic
**Management System for Feng Shui Stones (Quản lý các loại đá phong thủy)**  
A web application built using Java JSP/Servlet following MVC architecture to manage feng shui stone products, categories, users, and contact requests.

## 🚀 6. How to Run Project

### Step 1: Import Database
- Open MySQL Workbench
- Import file:
  /database/ddl/database.sql


---

### Step 2: Configure Server
- Setup Apache Tomcat 9 in IntelliJ IDEA
- Add project to Tomcat deployment

---

### Step 3: Run Project
- Click **Debug / Run**
- Open browser:
  http://localhost:8080

- Redirect to **Dashboard**

    - 🪨 Stones (Products)
